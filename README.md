# Easy-Redis

#### 项目介绍
基于redisson更高层次的封装、支持注解式分布式锁、发布订阅、消息有序队列、重试、工具类等


极速开始
-------------------------------------
以下例子基于Spring Boot

### 第一步：添加Maven依赖

直接添加以下maven依赖即可

```xml
<dependency>
    <groupId>io.github.javpower</groupId>
    <artifactId>easy-redis-spring-boot-starter</artifactId>
    <version>3.9.1</version>
</dependency>
```
